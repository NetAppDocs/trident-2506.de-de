.Bevor Sie beginnen
Stellen Sie sicher, dass die Gültigkeitsdauer des AWS-Sitzungstokens für alle länger laufenden S3-Backup-Vorgänge ausreichend ist.  Wenn das Token während des Sicherungsvorgangs abläuft, kann der Vorgang fehlschlagen.

* Siehe die https://docs.aws.amazon.com/STS/latest/APIReference/API_GetSessionToken.html["AWS API-Dokumentation"^] Weitere Informationen zum Prüfen des Ablaufs des aktuellen Sitzungstokens finden Sie hier.
* Siehe die https://docs.aws.amazon.com/IAM/latest/UserGuide/id_credentials_temp_use-resources.html["AWS IAM-Dokumentation"^] Weitere Informationen zu Anmeldeinformationen für AWS-Ressourcen finden Sie hier.
